# 🚗 STM32 智能小车 — 大一项目

> 大一第一个 STM32 项目，全部代码手敲完成。

基于 STM32F103C8T6 (HAL库)，实现了蓝牙遥控 + 5路循迹 + 超声波避障。

## 功能

- **蓝牙遥控**：`E`前进 / `H`后退 / `L`左转 / `R`右转 / `S`停止 / `X`旋转
- **5路循迹**：红外传感器阵列，黑线识别，自动纠偏
- **超声波避障**：HC-SR04 实时测距

## 代码结构

```
User/
├── USART.c/h      蓝牙串口指令解析
├── qudong.c/h     电机驱动（8个运动函数：前进/后退/左转/右转/停止/旋转/左旋/右旋/慢左）
├── xunji.c/h      5路循迹传感器读取 + 自动纠偏逻辑
└── bizhang.c/h    超声波测距（HC-SR04）
```

## 硬件

| 模块 | 说明 |
|------|------|
| STM32F103C8T6 | 主控 |
| HC-05 | 蓝牙模块 |
| HC-SR04 | 超声波测距 |
| 5路红外传感器 | 循迹 |
| L298N | 电机驱动 |
